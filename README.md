# Simulacija-nogometne-lige

Koda, ki simulira nogometno ligo. Uporabi se tako, da se za vsako ekipo določi parametre s "sliderjem" (kasneje bo liga predpripravljena in se bo samo pognala z gumbom) ter se doda ekipe. Ko so ekipe dodane, se določi število tekem med ekipami in se generira ligo. Spodaj se določi število željenih simulacij in se simulira liga.
Statistična analiza sledi kmalu.
